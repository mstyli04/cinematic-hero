  • 03

    NEURAL OPTION PRICER

     DEMO GITHUB

    A neural network that prices options by learning the Black–Scholes equation directly, with no market data involved. Training streams to a live dashboard where you can watch the price surface take shape and compare it against the exact solution in real time. It also handles the American put, which has no exact formula — the network works out the early-exercise boundary itself, checked against a traditional finite-difference solver.

    • PyTorch
    • FastAPI
    • three.js

    • Next.js
